### This is an ECS Demo based on 
https://github.com/solo-io/ecs-demo

1. Install aws cli

```
  sudo apt update
  sudo apt install -y unzip curl
  curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
  unzip awscliv2.zip 
  cd aws
  sudo ./install 
  aws version
  aws configure sso
```
